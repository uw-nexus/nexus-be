# NEXUS UW - Backend

Projects platform between non-profit organizations (NPO) and students.

## Getting Started
Install dependencies.
```sh
npm install
```

Define your environment variables in a `.env` file, in KEY=VALUE format.

| ENV | Description |
| --- | ----------- |
| PORT | Port exposed to serve backend |

## Run Intructions

### Development
```sh
npm run dev
```

### Production
```sh
npm start
```
